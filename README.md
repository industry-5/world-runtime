# World Runtime

_Last updated: 2026-03-23 (America/Chicago)_

World Runtime is a public portfolio for a runtime model built around explicit world state, append-only events, deterministic replay, simulation before action, and policy-governed decisions.

This repository is intentionally documentation-only. It is the public front door for the concept, the adapter portfolio, and the domain examples that show where the model fits.

World Runtime is a product of **INDUSTRY 5, Inc.**

## What It Helps You Explore

World Runtime is aimed at systems where teams need:

- explicit world state
- append-only event history
- deterministic replay and projection
- policy-visible decisions
- simulation before action

The public examples in this repo focus on governance-heavy, operator-facing, and agent-facing domains where consequences cascade and decisions need evidence.

## Public Adapter Portfolio

The current portfolio includes these public tracks:

- `adapter-supply-network`
- `adapter-air-traffic`
- `adapter-semantic-system`
- `adapter-power-grid`
- `adapter-city-ops`
- `adapter-lab-science`
- `adapter-market-micro`
- `adapter-multiplayer-game`
- `adapter-autonomous-vehicle`
- `adapter-multi-agent-ai`
- `adapter-open-agent-world`
- `adapter-digital-twin`

`adapter-digital-twin` is presented as a host-bound overlay across public host domains, not as a standalone world.
