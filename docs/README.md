# Documentation Hub

_Last updated: 2026-03-23 (America/Chicago)_

This folder is the lightweight index for the public World Runtime documentation set.

The public repo is intentionally documentation-only. Use it to understand the portfolio story, compare domain tracks, and see how the runtime model maps onto different kinds of governed systems.

## Public Repo Boundaries

- This repo is a public-facing documentation surface, not a runnable distribution.
- Process-tracking materials, internal handoff prompts, and validation infrastructure are intentionally excluded.
- The adapter pages describe portfolio intent and domain fit, not a public commitment that every underlying implementation surface is published here.
