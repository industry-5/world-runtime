# Documentation Hub

_Last updated: 2026-03-14 (America/Chicago)_

This hub is the fastest way to navigate the World Runtime docs by intent instead of by folder layout.

The docs surface includes both support-committed runtime material and best-effort showcase/storytelling material; use [docs/SUPPORT_POLICY.md](SUPPORT_POLICY.md) when you need the authoritative support boundary.

Docs taxonomy for this repo:

- `world-runtime` = core runtime and support boundary
- `labs/world_game_studio_next` = primary showcase app and demo path
- `adapters/world_game` = World Game domain package and runtime source of truth
- `labs/world_game_studio` = retired legacy surface
