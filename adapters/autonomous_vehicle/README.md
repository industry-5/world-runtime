# Autonomous Vehicle

_Last updated: 2026-03-23 (America/Chicago)_

`adapter-autonomous-vehicle` shows how World Runtime fits motion-planning systems where safety boundaries, intervention logic, and least-bad choices need to be explicit.

## What This Track Demonstrates

- dynamic planning under hard safety constraints
- reviewed fallback and intervention decisions
- replayable evidence for maneuver choices
- simulation of alternate safety-limited actions

## Why It Matters

Physical-space reasoning creates cases where every option has cost. This track highlights how the runtime model can keep those tradeoffs explicit instead of hiding them behind a single opaque action.

## Boundaries

- this public page describes the domain story only
- it does not publish internal process history or validation commands
- the track stays focused on motion safety and intervention, not broad autonomy platform scope
