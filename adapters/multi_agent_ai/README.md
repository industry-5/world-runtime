# Multi-Agent AI

_Last updated: 2026-03-23 (America/Chicago)_

`adapter-multi-agent-ai` shows how World Runtime fits governed multi-agent systems where agents can propose, branch, and coordinate, but policy still determines what becomes real.

## What This Track Demonstrates

- agent proposal and review flows
- branching coordination paths
- explicit governance over agent-generated action
- replayable evidence for why one branch was accepted over another

## Why It Matters

Agent systems become easier to trust when they are treated as contributors to a governed world model rather than as an opaque command surface. This track highlights that review-oriented posture.

## Boundaries

- this public page describes the domain story only
- it does not publish internal handoff files or validation infrastructure
- the track stays focused on governed coordination rather than broad agent-platform scope
