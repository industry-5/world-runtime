# Power Grid

_Last updated: 2026-03-23 (America/Chicago)_

`adapter-power-grid` shows how World Runtime fits critical-infrastructure systems where balancing, contingencies, and least-bad choices must be handled with explicit evidence.

## What This Track Demonstrates

- balancing and contingency response
- simulation of cascading infrastructure effects
- policy-visible intervention under hard constraints
- replayable evidence for high-consequence decisions

## Why It Matters

Power systems are a strong test for deterministic replay and simulation because local failures can propagate quickly. The runtime model is useful when operators need to compare options, understand consequences, and preserve a clear record of why a choice was made.
