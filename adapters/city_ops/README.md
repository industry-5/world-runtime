# City Ops

_Last updated: 2026-03-23 (America/Chicago)_

`adapter-city-ops` shows how World Runtime fits cross-agency civic coordination where transportation, utilities, and public response need a shared operating picture.

## What This Track Demonstrates

- multi-agency incident coordination
- shared-state decisions across overlapping civic systems
- simulation of response options before action
- explicit visibility into tradeoffs between competing public needs

## Why It Matters

City operations are rarely isolated inside one department. This track shows how a common world model can support coordinated decisions when outages, traffic, public safety, and service restoration all interact.

## Boundaries

- this public page describes the domain story only
- it does not publish internal process materials or validation commands
- this track also serves as a public host surface for `adapter-digital-twin`
